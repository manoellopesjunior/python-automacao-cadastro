# Automação de Cadastro de Produtos com Python

Projeto desenvolvido durante o **Intensivão de Python da Hashtag Programação** com o objetivo de automatizar o processo de cadastro de produtos em um sistema web.

## 📌 Descrição

Este script utiliza **Python e PyAutoGUI** para automatizar tarefas repetitivas no computador, como abrir um navegador, acessar um sistema e cadastrar produtos automaticamente a partir de uma base de dados.

A automação reduz o tempo gasto em tarefas manuais e diminui a chance de erros humanos durante o processo de cadastro.

## 🚀 Tecnologias utilizadas

- Python
- PyAutoGUI
- Pandas

## 📂 Estrutura do projeto

Python_Powerup
    auxiliar.py         # Funções auxiliares utilizadas no projeto
    codigo.py           # Script principal da automação
    produtos.csv        # Base de dados com os produtos
    requirements.txt    # Bibliotecas necessárias para executar o projeto


## 📊 Funcionamento

O script realiza os seguintes passos:

1. Abre o navegador
2. Acessa o sistema de cadastro
3. Realiza o login automaticamente
4. Lê os dados dos produtos a partir do arquivo `produtos.csv`
5. Preenche os campos no sistema
6. Realiza o cadastro de cada produto automaticamente
